Ubuntu 下通过Wine安装阿里旺旺并解决中文乱码

[日期：2016-05-08] 来源：Linux公社  作者：Linux [字体：大 中 小]

现在 Wine 功能越来越完善，也越来越强大了，在Ubuntu 下用Wine来安装阿里旺旺其实是一件很容易的事情，你需要做的事情仅仅是下载AliIM2015_taobao(8.60.00C).exe，下载地址：http://download.taobaocdn.com/wangwang/AliIM2015_taobao(8.60.00C).exe

本文Linux测试环境为Ubuntu 15.10，同样Ubuntu 16.04也适用。wine 为1.9.6版本。

首先安装好Win3，安装方法见 Ubuntu 15.10 通过官方PPA安装Wine 1.8稳定版 http://www.linuxidc.com/Linux/2015-12/126722.htm

开始安装：AliIM2015_taobao(8.60.00C).exe，右键选择打开，见下图：

打开winecfg配置下Wine，在Applications选项页下，选择Windows Version为Win 200,xp 或者vista。

安装好后桌面上就有阿里旺旺的图标。双击阿里旺旺图标。看，是不是运行起来了，不需要任何多余的配置。

见上图，这时是乱码的。在浏览器中仍然无法从卖家的联系图标链接直接激活阿里旺旺。所以你还得继续完成以下步骤：

1、在 ~/.local/share/applications 下创建一个名为 wine-extension-aliim.desktop 的文件，填入以下内容：

Type=Application
 Name=AliWangWang
 MimeType=x-scheme-handler/aliim;
 NoDisplay=true
 StartupNotify=true
 Icon=A767_AliIM.0
 wine "C:\Program Files\AliWangWang\8.60.00C\wwcmd.exe" %u

方法见下图

2、打开终端，输入命令：（见上图）

xdg-mime default wine-extension-aliim.desktop x-scheme-handler/aliim

3、为了解决中文乱码问题，也为了让 Windows 应用程序看上去更美观，所以需要 Windows 下面的字体。

我这里选择simsun.ttc，看下图，点击右上角的安装，不用什么复制到 ~/.wine/drive_c/windows/fonts/ 目录中。但是.ttc 是许多 Windows 应用默认使用 simfang.ttc 字体。

 

这是再打开就不再乱码了，见最上面截图。
